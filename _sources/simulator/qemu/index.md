# QEMU

QEMU（Quick Emulator）是一个开源的虚拟机监控程序（hypervisor）和仿真器，广泛用于虚拟化、系统仿真和开发调试。它能够模拟多种硬件架构，并在这些架构上运行不同的操作系统。QEMU 的功能非常强大，既可以作为系统仿真器运行，也可以作为虚拟化工具与 KVM（Kernel-based Virtual Machine）结合使用，提供接近原生性能的虚拟化环境。
